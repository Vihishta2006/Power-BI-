# Power BI:
**Data Professional Survey Dashboard**

This file has one data table called "Data Professional Survey", holding responses from people working in data-related jobs. 
This is a survey results dashboard, summarizing who data professionals are (country, role, age), what tools/languages they like, and how happy they are with pay and work-life balance.

**Video Game Sales Dashboard**

This file has one data table called "vgsales" with global video game sales figures. It has 2 report pages:

First page has -> # A column chart comparing Japan sales vs North America sales, broken down by game Genre
#A donut chart showing game rank split by Genre
# A table listing total global sales by Publisher

Second page has -> A combo chart (line + stacked columns) showing "Other region" sales and rank by game Name
A line chart comparing NA, EU, and JP sales across genres
A treemap of total global sales by Genre
A pivot table breaking down rank by Platform and Genre

These are the simple dashboards analyzing video game sales data across regions (NA, EU, JP, and others), genres, publishers, and platforms.
